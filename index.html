Getting Started in Open Source with Python

Brought to you by PyLadies Chicago & Write/Speak/Code

Passionate about Open Source?

Join us on July 25th for "Getting Started in Open Source w/ Python" where you will be turned into an Open Source contributor! From git to licensing, you'll learn all about open source logistics and culture. Learn where to find projects and how to determine which projects are best for you. Get the low-down on how best to contribute and engage with open source maintainers.
We will have projects and Python mentors for you to work on and support you as you make your first 1st open source contribution. Publish and contribute to open source by the end of the day!
Click here to register!

This event is meant for those that have experience in Python and Git.


Schedule

Morning: A light breakfast will be provided and an Intro to Open Source talk by Write/Speak/Code.

Lunch: Complimentary lunch!

Afternoon: Code on Python projects. Mentors will be available to answer questions. Example projects targeted: Django, Flask, Census Reporter, Tarbell, Chicago Python User Group website, PyLadies website and more!
